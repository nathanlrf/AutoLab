AutoLab
=======

本科毕业设计——基于MATLAB的自控实验仿真实验

主要登陆计算机：2号楼

这个版本库主要用来记录我的C语言驱动代码

驱动由四部分构成：

1、AD模块：最为复杂，输入为16路

2、DA模块：输出为2路

3、DI、DO模块：较为简单，都只有1路输入和输出，数据格式为16 bits
